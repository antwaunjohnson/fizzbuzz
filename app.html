FIZZ BUZZ: A Coding Project

Fizz Buzz'n the count! A javascript project that prints a set of numbers. If the number is a multiple of 3 "Fizz" is displayed. If the number is a multiple of 5 "Buzz" is displayed. Finally, numbers that are multiples of both will display "FizzBuzz".

Instructions

Choose a starting value and an ending value. Press the button to Fizz Buzz the count.
