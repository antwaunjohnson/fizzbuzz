                            function getValues() {
                                let fizzValue = document.getElementById("fizzValue").value;
                                let buzzValue = document.getElementById("buzzValue").value;
                              
                                fizzValue = parseInt(fizzValue);
                                buzzValue = parseInt(buzzValue);
                              
                                if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {
                                  let fbArray = fizzBuzzC(fizzValue, buzzValue);
                                  displayData(fbArray);
                                } else {
                                  alert("You must enter integers to Fizz Buzz the count!");
                                }
                              }
                              
                              function fizzBuzz(fizzValue, buzzValue) {
                                let returnArray = [];
                              
                                // loop from 1 to 100
                                for (let i = 1; i <= 100; i++) {
                                  if (i % fizzValue == 0 && i % buzzValue == 0) {
                                    returnArray.push("FizzBuzz");
                                  } else if (i % fizzValue == 0) {
                                    returnArray.push("Fizz");
                                  } else if (i % buzzValue == 0) {
                                    returnArray.push("Buzz");
                                  } else {
                                    returnArray.push(i);
                                  }
                                }
                                return returnArray;
                              }

The code is structured in three functions

getValues

The getValues functions is our main function. We begin my getting the input from the user. The users input is stored in two variables fizzValue and buzzValue. This is done my using the document method 'getElementById' to find each element with the id or fizzValue or buzzValue. When we receive the input we must make sure that the values are numeric. This is done by using the parseInt function. We set our fizzValue and buzzValue equal to parseInt then pass the values as arguments.

Once we are sure that are values are numbers we use an if statement to execute the FizzBuzz program. We begin the if statement checking the fizzValue and buzzValue simultaneously. This can be done by passing each value to the Number.isInteger() method. Inside of our if statement we create the fbArray variable and set it equal to one of our fizzBuzz function, passing the fizzValue and buzzValue as arguments. Finally, the displayData function is called and passed the fbArray variable as argument. Finally, if either of the values are not a number, the else block is executed which alerts the user "You must enter integers to Fizz Buzz the count!"

The fizzBuzz functions

Each of the fizzBuzz functions take the fizzValue and buzzValue as arguments. They each begin by setting the variable returnArray to an empty array. For each for loop we set i = 1 as long as i is less than or equal to 100, i++ is used to increment the count until the loop is completed

Each of the functions uses a different method to reach the "FizzBuzz" objective.

Function 1
fizzBuzz uses a for loop. We loop over each number first checking to see if it is divisible by or fizzValue and buzzValue. If both values are divisible, "FizzBuzz" is pushed into the array. Next we check to see if the number is divisible by the fizzValue. If this is true, "Fizz" is added to the array in place of the number. Next we check to see if the number is divisible by the buzzValue. If this is true, "Buzz" is added to the array in place of the number. Finally, if the number is not divisible by either the fizzValue or the buzzValue, the number is added to the array. Once we have looped through all the numbers, the completed array is returned.

Function 2

Before we being our for loop in the fizzBuzzB function, we create two variables Fizz and Buzz setting their values to false. With the for loop Fizz is equal to i % fizzValue == 0 while Buzz is equal to i % buzzVAlue == 0 Next we use a switch statement to check if each case within our switch statement is true. The first case checks if both values are true. If both values are true, "FizzBuzz" is added to the array. If the Fizz case evaluates to true, "Fizz" is added to the array. If the Buzz case evaluates to true, "Buzz" is added to the array. Finally, if none of the cases within the switch statement evaluate to true, the default is to add the number to the array. Once we have looped through all the numbers, the completed array is returned.

Function 3

Finally, fizzBuzzC uses the conditional (ternary) operator. We create the variable value and set it equal to our ternary operator. Each expression checks to see whether the number divisible by the fizzValue or buzzValue is equal to zero if the number is not divisible by either expression the value is set to an empty string. If the value is divisible by the fizzValue and buzzValue, concatenation is used to add "FizzBuzz" into the array. If the value is only divisible by fizzValue or buzzValue, either "Fizz" or "Buzz" is added to the array. If the value is not divisible by the fizzValue or buzzValue, the value is set equal to i, then we add the value into the array. Once we have looped through all the numbers, the completed array is returned.

displayData

The displayData function takes fbArray as an argument. We use the document method 'getElementById' to find the element with the id of 'results' and set it equal to tableBody. We also use the same method to set tableRow equal to the element that has the id of 'fbTemplate'. What is unique about our html element that holds the id of 'fbtemplate' is the content template element. According to The Mozilla docs "The template element is a mechanism for holding html that is not rendered immediately when a page is loaded but may be instantiated subsequently during runtime using JavaScript". In this function we set the index equal to 0. As long as the index is less than the length of fbArray, we use index += 5 to iterate by 5. Within the for loop we use the document method importNode to create a document fragment from our content template element and set it equal totableRow. This method takes two arguments, templateRow.content and true to ensure that all the descendants of the node are copied. Next we set the variable rowCols equal to all of the td elements using the querySelectorAll method. Since we only want to have 5 elements in each row, we add a the classList method to dynamically add a class of .Fizz, .Buzz, or .FizzBuzz to color to either of the elements that display Fizz, Buzz or FizzBuzz. We also set the textContent of each element to the index of fbArray. Finally we use the appendChild method and pass it the argument of tableRow

resetForm
Finally, the reset form function is used to reset the form to its default values.
